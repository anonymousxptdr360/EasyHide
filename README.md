# EasyHide - Ad Remover for Shipping Labels (Batch Processing)
**EasyHide** is a powerful, client-side web tool designed to quickly hide unwanted
advertisements or specific areas on shipping labels (Bordereaux) and documents. It supports
batch processing for both images and PDF files.
### Try the Live Version
You can use the official web interface here:
 [https://stopbordo.odoo.com/bordereau](https://stopbordo.odoo.com/bordereau)
### Key Features
* **Batch Processing:** Upload multiple files at once and apply the same mask to all of
them.
* **Hybrid Support:** Works seamlessly with `.pdf`, `.png`, `.jpg`, and `.jpeg`.
* **Interactive UI:** Select a quadrant and fine-tune the hiding zone with a
draggable/resizable rectangle.
* **Privacy First:** Everything happens in your browser. No files are uploaded to any server.
* **Custom Output:** Export as original format or convert everything to PDF.
### How to use
1. **Select a Quadrant:** Choose the general area where the ad is located.
2. **Upload Files:** Select one or more shipping labels.
3. **Adjust:** Use the first file to precisely resize the white mask.
4. **Process:** Click "Apply to All" to download your cleaned documents instantly.
